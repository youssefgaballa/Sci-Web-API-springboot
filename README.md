# Sci-Web API - Spring Boot


### How to run spring-backend
Create .env file in root directory and add the following environment variables:
```
POSTGRES_USERNAME: <your_username>
POSTGRES_PASSWORD: <your_password>
POSTGRES_DB: <your_database_name>
POSTGRES_HOST_PORT: <your_host_port>
POSTGRES_CONTAINER_PORT: <your_container_port>
```
